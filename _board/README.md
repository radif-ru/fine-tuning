# Доска задач

Эта директория содержит управление проектом: спринты, задачи и процессы.

## Структура

| Файл | Назначение |
|------|------------|
| `README.md` | Этот файл — описание структуры доски |
| `plan.md` | Индекс спринтов (динамические данные) |
| `process.md` | Процесс выполнения спринтов и задач, легенды, шаблоны |
| `progress.txt` | Ad-hoc заметки, текущий контекст |
| `sprints/*.md` | Файлы отдельных спринтов |

## Навигация

- 📋 **План** — [`plan.md`](./plan.md)
- 🔄 **Процесс** — [`process.md`](./process.md)
- 📊 **Прогресс** — [`progress.txt`](./progress.txt)
- 🏃 **Спринты** — [`sprints/`](./sprints/)

## Как пользоваться

1. Читай `process.md` перед началом работы
2. Смотри текущий спринт в `sprints/`
3. Обновляй `progress.txt` по мере работы
4. При закрытии спринта — обновляй `plan.md`

## Текущий спринт

См. [`plan.md`](./plan.md) § «Индекс спринтов» для актуального статуса.
